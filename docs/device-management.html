Device Management

Device Registration​

Manual Device Addition (Not Recommended)​

  1. Click the Device button.
  2. In the device addition form, select the project where the device is located, the device name, and the device SN code.
  3. Click the Create button to complete the creation of the device.
warning

For security reasons, manually adding a device requires a complete 32-digit SN code, which can be obtained by scanning the QR code on the back of the device.

Importing Devices Through Excel (Recommended)​

  1. Click the Import button.
  2. In the popup, click the Download button to get the Excel device import template.
  3. Fill out the Excel sheet (if the project has not been created, the system will automatically create one).
  4. Upload the Excel sheet.
  5. Click the Confirm button in the popup to complete the device import.

Importing Devices Through YoLink APP (Recommended)​

  1. Click the Inventory button to enter the inventory management page.
  2. On the inventory management page, click the YoLink button.
  3. After redirecting to the YoLink authorization page, log in to your YoLink account. Raedius will get the device list from the YoLink account and add it to the inventory.
  4. On the inventory management page, find the device you want to add and click the Bind button.
  5. In the device binding popup, select the project where the device is located.
  6. Click the Bind button to complete the device registration.
info

Devices in the inventory are in an inactive state, the system will not record device data, and inventory devices will not occupy the project device quota.

Device View​

Table View​

info

Device status description:

  • Red: The device is in an alarm state (alarm defined in personalization or custom alarm).
  • Gray: The device is offline.
  • Black: The device is in a normal state.

Card View​

info

Device status description:

  • Alternating card deletion: The device is in offline/alarm state.

Device Editing​

  1. Click the Operation button and select Edit in the dropdown menu.
  2. Modify the device information in the device information popup.
  3. Click the Update button to complete the device information change.
info

It is not possible to change the device's project through the device update feature. To change the device's project, please use the migration feature.

Device Deletion​

  1. Click the Operation button and select Delete in the dropdown menu.
  2. In the device confirmation deletion popup, click the Delete button to delete the device.

Device Historical Data Export​

  1. Select the devices for which you want to export data.
  2. Click the export button in the operation bar.
  3. In the data export confirmation popup, select the data export range (not exceeding 12 months).
  4. Click the Export button in the popup to complete the data export.

Device QR Code Export​

  1. Select the devices for which you want to export QR codes.
  2. Click the export QR code button in the operation bar.
  3. In the QR code export confirmation popup, enter the email address to receive the QR code data.
  4. Click the Send button in the popup to complete the QR code export.
warning

For data security reasons, exporting QR codes for devices imported through the YoLink APP is not supported.

Device Migration​

  1. Select the devices that need to be migrated.
  2. Click the export Migration button in the operation bar.
  3. In the popup, select the new project for the devices.
  4. Click the Confirm button in the popup to complete the device migration.
warning

Please note: Device migration may affect the following features:

  • Device Grouping
  • Action Configuration

Device Charts​

Device Personalization​

To lower the user threshold, Raedius has defined default alarm behaviors and lock behaviors through device personalization. You can make adjustments by modifying the device personalization.

Device Action Configuration​

Users can define specific actions to be executed when the device is in a certain state through device action configuration, similar to device automation. Device-level action configuration only applies to the current device. In addition to device-level action configuration, Raedius also provides group-level and project-level action configurations. For more about action configuration, refer to the action configuration documentation.

Device Activities​

Device activities record the device's activity history, such as device registration, status changes (online/offline/alarm/temperature changes, etc).
